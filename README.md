# Build instructions for Ubuntu

# Install Gtk
```console
$ sudo apt-get install libgirepository1.0-dev libwebkit2gtk-4.0-dev libgtksourceview-3.0-dev
```

# Install Sdl2
```console
$ sudo apt-get install libsdl2-dev
```

# Build with stack
```console
$ stack build
```

# Usage
```console
$ stack exec pure-nes
```

# Run tests
```console
$ stack test
```
